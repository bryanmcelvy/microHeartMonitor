void SPI_Init ( void  )

Initialize SSI0 to act as an SPI Controller (AKA Master) in mode 0.

The bit rate BR is set using the (positive, even-numbered) clock prescale divisor CPSDVSR and the SCR field in the SSI Control 0 (CR0) register:

The ILI9341 driver has a min. read cycle of 150 [ns] and a min. write cycle of 100 [ns], so the bit rate BR is set to be equal to the bus frequency ( ) divided by 8, allowing a bit rate of 10 [MHz], or a period of 100 [ns].
